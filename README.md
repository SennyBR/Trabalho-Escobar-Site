# Requisito

- Php 8.2
- Laravel 11
- Apache
- MySQL

# Como Rodar

- Use a função
  - composer install
- Abaixe o arquivo
- Direcione para uma pasta chamada carro-de-orda
- Adicione o DB em phpmyadmin
- Após isso realizar a função
  - php artisan migrate
  - php artisan serve
